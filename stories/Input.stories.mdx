## Behavior
### Default input/ Prefix and suffix input

1. Click on the component to focus
2. The placeholder will move up and behave like a label
3. Type what the label suggests you to
4. If you don't fill the input correctly, the input will show a error help message,
changing the component for error state
5. Click outside the component to unfocus

### Required Input

1. Click on the component to focus
2. The placeholder will move up and behave like a label
3. Type what the label suggests you to
4. If you don't fill the input correctly,
the input will show a error help message, changing the component for error state
5. Click outside the component to unfocus
6. If you don't fill the input at all, the input will show a required help
message after you unfocus it, changing the component for error state.
