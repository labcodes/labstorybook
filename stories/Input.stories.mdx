#### Behavior
<div class="columns">
  <div class="column">
    <h6>Default input/ Prefix and suffix input</h6>
    <ol>
      <li>Click on the component to focus </li>
      <li>The placeholder will move up and behave like a label</li>
      <li>Type what the label suggests you to</li>
      <li>If you don't fill the input correctly, the input will show a error help message, changing the component for error state</li>
      <li>Click outside the component to unfocus</li>
    </ol>
  </div>
  <div class="column">
    <h6>Required Input</h6>
    <ol>
      <li>Click on the component to focus</li>
      <li>The placeholder will move up and behave like a label</li>
      <li>Type what the label suggests you to</li>
      <li>If you don't fill the input correctly,the input will show a error help message, changing the component for error state</li>
      <li>Click outside the component to unfocus</li>
      <li>If you don't fill the input at all, the input will show a required help message after you unfocus it,
      changing the component for error state.</li>
    </ol>
  </div>
</div>
