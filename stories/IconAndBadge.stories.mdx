  <h2>Overview</h2>
  <p>Will display what is this component about with a simple intro text like: "Radio is a type of selector that is used when we have a list of options, and only one option can be selected."</p>
  
  <p>The <code>Icon</code> and <code>Badge</code> components have similar behavior.</p>
  <p>Both receive two inputs as props, where <code>type</code> indicates which symbol to show, while <code>color</code> defines its color. <code>Badge</code> receives an extra prop <code>wrapperColor</code>, since the background color is different from the symbol color. Also, the <code>size</code> option is available only for <code>Icon</code></p>
