<h1>Selectors</h1>
<h2>General Usages</h2>
<div class="columns">
    <div class="column">
        <p>Radio and Checkboxes are selection controls that allows the user to select options. 
        They should be used in situations that the user needs to decide or declare preferences by choosing options on the screen.
        We can have these 2 types of selectors:
        </p> 
    </div>
    <div class="column">
        <p>
            <strong>Radio</strong>
            <br />
            This type of selector is used when we have a list of options where only one option can be selected.
        </p> 
        <p>
            <strong>Checkbox</strong>
            <br />
            This type of selector is used when we have a list of options that can be multiple selected or don't have 
            a selected option at all.
        </p> 
    </div>
</div>
