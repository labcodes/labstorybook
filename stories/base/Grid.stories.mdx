  <h2>Container</h2>
  <p>Besides the columns, the <code>container</code> is another important element to define layout.</p>
  <p>It is a simple utility element that allows you to center content on larger viewports. Using the class <code>container</code> on a div will limit its width and center it horizontally.</p>
