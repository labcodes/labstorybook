## Style

Tags have two styles settings. Use `outline` to use borders instead of a filled background. Use `skin="vivid"` to display more vivid colors.

##### Default tag style
If no color setting is used, the tag will be displayed with a mineral fill background using `skin="pale"`.

##### Disabled
This prop is used if a tag action is disabled. Therefore, Simple Tags cannot receive this prop.

All disabled styles remain the same regardless of any property other than `outline`.

## Color
The `color` prop can be set to any tag.

##### Theme
Tag's default color is `mineral` but it can also be set to teal or purple from the theme settings or by the user with `color="teal"` or `color="purple"`.

All disabled styles remains unchanged regardless of any value used.
