  <p>Checkboxes are selection controls that allows the user to select options.
  They should be used when we have a list of options that can be multiple selected.
  </p>

  ##### Acessibility
  Striving to keep our forms accessible, we recommend to use this type of component by using the `fieldset` tag to nest a group of
  selectors and the `legend` tag to identify the heading of the group.
  These elements works indicating to screen readers that a group of form fields relate to each other.
