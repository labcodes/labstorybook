# General usage

This component is a form field that allows user to type text information.
We can have 4 types of text inputs:

## Default Input

It’s a standard text input. Can be used for enter a name, phone number, street address, email address, etc.

<Preview>
  <Story name="Default Input">
  <div className="input-wrapper">
    <ExampleInputs id="name" placeholder="  "></ExampleInputs>
    <label for="name">Label</label>
  </div>
  </Story>
</Preview>

## Required Input

It’s a default input that is required to be filled. It has a helper message to inform if a required input text is missing.

<Preview>
  <Story name="Required Input">
  <div className="input-wrapper">
    <span class="required"></span>
    <ExampleInputs className="text-input" id="name" placeholder="  "></ExampleInputs>
    <label for="name">Label</label>
  </div>
  </Story>
</Preview>
